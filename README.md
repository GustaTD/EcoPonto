# EcoPonto

## Nome do projeto
EcoPonto

## Integrante
Gustavo Toledo Denig

## Descrição resumida do sistema
O EcoPonto é uma aplicação acadêmica voltada para a cidade de Cascavel - PR, com o objetivo de mostrar e organizar pontos de coleta seletiva.

O sistema permite cadastrar, consultar, editar e excluir pontos de coleta de materiais como baterias, eletrônicos, óleo, remédios, recicláveis e outros itens.

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript
- LocalStorage
- Leaflet
- OpenStreetMap
- Git/GitHub

## Funcionalidades implementadas
- Tela inicial do sistema
- Mapa com pontos de coleta
- Cadastro de ponto de coleta
- Listagem e consulta dos pontos cadastrados
- Filtro de busca por nome, bairro ou tipo de coleta
- Edição de ponto de coleta
- Exclusão de ponto de coleta
- Persistência dos dados no navegador usando LocalStorage

## Instruções para execução
1. Baixe ou clone o repositório.
2. Acesse a pasta `codigo-fonte`.
3. Abra o arquivo `index.html` em um navegador.
4. Para testar o fluxo completo:
   - Cadastre um novo ponto.
   - Consulte o ponto na listagem.
   - Edite as informações do ponto.
   - Exclua o ponto cadastrado.
